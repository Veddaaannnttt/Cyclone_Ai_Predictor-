# Cyclone Intensity Prediction AI — North Indian Ocean
End-to-end ML pipeline for predicting tropical cyclone intensity using IBTrACS best-track data and ensemble ML (Random Forest + XGBoost + LightGBM).

## Metrics
- Regression MAE: ~2 knots | R²: ~0.94
- Classification Accuracy: ~86% | F1: ~85%

## Run
Open `Cyclone_Prediction_NIO.ipynb` in Google Colab and run all cells.
